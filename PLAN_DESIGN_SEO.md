# Plan de Diseño, SEO y Contenidos - Autolock

## 1. Sitemap Propuesto (Estructura de Conversión)

La estructura está diseñada para minimizar clics y guiar al usuario a la llamada o contacto.

### Main Pages
- **Home**: Página de aterrizaje principal. Objetivo: Llamada inmediata o WhatsApp.
- **Servicios**: Hub central con tarjetas de todos los servicios.
  - *Subpáginas (Futuras)*: /services/lost-keys, /services/key-repair, etc.
- **ECU Remapping**: Página dedicada (Landing específica para este servicio de alto valor).
- **TPMS**: Página dedicada para sensores de presión.
- **Cobertura**: Mapa y lista de localidades (SEO Local).
- **Contacto**: Formulario simple + Datos de contacto.

### Utility Pages
- **FAQ**: Preguntas frecuentes para reducir fricción.
- **Galería**: Muestra de trabajos (confianza).
- **Aviso Legal / Cookies**.

## 3. UI Kit & Estilo

### Colores
- **Primario**: Rojo Automoción (`#D32F2F`) -> Acción, urgencia.
- **Fondo**: Blanco (`#FFFFFF`) o Gris Muy Claro (`#F8F9FA`) -> Limpieza.
- **Texto**: Gris Oscuro (`#1F2937`) -> Legibilidad.
- **Botón WhatsApp**: Verde (`#25D366`).

### Tipografía
- **Titulares y Cuerpo**: `Inter` o `Roboto`. Moderna, limpia, legible en móvil.

### Componentes Clave
- **Botones**: Esquinas redondeadas (8px), sombra suave.
- **Cards**: Fondo blanco, borde sutil, efecto "lift" al hacer hover.
- **Barra Móvil Inferior**: Sticky, dividida en "Llamar" (Rojo) y "Presupuesto" (Gris).
